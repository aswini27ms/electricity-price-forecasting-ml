# Electricity Price Forecasting and Renewable Impact Analysis using ML

## Project Description

This project focuses on forecasting electricity prices using machine learning techniques and analyzing the impact of renewable energy sources such as solar and wind power on electricity market prices. The system uses real-world Spain electricity market data and provides accurate predictions through a Streamlit-based web application.

---

## Features

- Electricity price forecasting using ML
- Renewable energy impact analysis
- Random Forest Regression model
- Explainable AI using feature importance
- Streamlit web application
- Interactive visualizations
- Real-time prediction interface
